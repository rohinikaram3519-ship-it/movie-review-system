# Movie Review Sentiment Analysis

## Description
This project analyzes movie reviews and predicts sentiment (positive/negative) using NLP techniques. It is trained on the IMDB dataset and also provides emotion scores like joy, fear, and sadness.

## Features
- Sentiment prediction (Positive / Negative)
- Emotion detection (joy, fear, sadness)
- Text preprocessing and cleaning
- Machine learning model implementation
